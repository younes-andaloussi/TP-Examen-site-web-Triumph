# TRIUMPH - La Legende Britannique

Site web statique consacre a l'histoire de Triumph Motorcycles et a l'evolution de ses modeles emblematiques.

Ce projet a ete realise dans le cadre du cours de developpement Web HTML5/CSS3 du bachelier en Informatique de Gestion.

## Site publie

Le site est accessible en ligne a l'adresse suivante :

[https://younes-andaloussi.github.io/TP-Examen-site-web-Triumph/index.html](https://younes-andaloussi.github.io/TP-Examen-site-web-Triumph/index.html)

## Presentation

L'objectif du projet est de concevoir un site web statique respectant les normes HTML5 et CSS3, avec une structure claire, une navigation coherente, une mise en forme CSS personnalisee et une organisation propre des fichiers.

Le theme choisi est l'histoire de Triumph Motorcycles. Le site presente l'evolution de la marque ainsi que plusieurs modeles importants :

- Bonneville
- Daytona
- Street Triple
- Speed Triple
- Tiger
- Rocket
- Thruxton
- Scrambler

Chaque page de modele retrace les principales generations de la moto, avec un contenu organise en articles, des images et un sommaire de navigation interne.

## Pages du site

- `index.html` : page d'accueil du site.
- `pages/histoire.html` : histoire generale de Triumph Motorcycles.
- `pages/nous-rejoindre.html` : formulaire de contact / adhesion.
- `pages/a_propos.html` : presentation du projet.
- `pages/motos/` : pages consacrees aux modeles Triumph.
